## Create Chat Completion

[OpenAI Chat Completion API](https://platform.openai.com/docs/api-reference/chat) compitable API with `gpt-4o` model.

```sh
curl https://openai.vivgrid.com/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $API_TOKEN" \
  -d '{
    "model": "gpt-4o",
    "messages": [
      {
        "role": "system",
        "content": "You are a helpful assistant."
      },
      {
        "role": "user",
        "content": "Hello!"
      }
    ]
  }'
```

### Authentication

`$API_TOKEN` can be found on your dashboard.
